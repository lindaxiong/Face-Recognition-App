# SmartBrain - Face Recognition Web App

Front End:
* HTML5
* CSS3
* Javascript + React

Back End:
* Node Server: Node.js, Express.js
* Database: PostgreSQL

## To create your own:
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. You must add your own API key in the `src/App.js` file to connect to Clarifai. 
You can grab Clarifai API key [here](https://www.clarifai.com/)
5. Deploy on Heroku
